Projects



Predicting Events with Twitter

For this project data mining techniques were used to collect real time twitter messages on topic such as earthquakes, weather, and flu. Then techniques such as sentiment analysis were used to predict the occurance of these topics.



Precision Education

For this project data analytics was used to identify areas where precision education could be implemented to aid the education of underrepresented groups.



Resmed

Resmed is a mobile application that can be used to aid the repair of sleep apnea machinery. Our team developed high fidelity prototypes and preformed user testing.



Multilingual Analysis Tool

This dashboard was created for a visual analysis of sentiment and semantic similarities between words in different languages.



Responsive Web Design

This responsive web design helps users identify a book they may be intrested in and where in the library they can locate it.



Transit Equity Dashboard

For this project we built a data dashboard that represents equity in the Pittsburgh transit system.



Fintor

Fintor is a project aimed at better understanding how young adults find financial support during the home buying process.



Interactive Course Website

For this project an interactive website was created to allow students to search and select courses.This is the redesign of Carnegie Mellon's existing site and works to solve user interface limitations



Museek

Museek is a conversational voice assistant that helps musicians train on their instruments. Museek is different from other voice assistants as it aims to provide fluid conversation with less of a transactional goal.
